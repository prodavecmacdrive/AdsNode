Final Export Tab (tab-final)

1. Navigation Bar and Status Bar

Final tab is active (Read-Only mode for previous tabs).

Top bar: Status "Playable approved and ready for download".

2. List of Builds (Ad Networks): A vertical list of ad network names. Next to each network is a checkbox.

Default state: All available builds (checkboxes) are checked.

Quick management: Text button-links above the list: Select All / Clear All. This solves the case when there are many networks, but the client only needs to download one — they click "Clear All" and check the needed checkbox.

Main action: A large prominent button at the bottom of the block "Download selected builds". Upon clicking, the system provides one unified ZIP archive containing the requested networks inside.

3. Chat Zone

Standard hierarchical log with chat at the very bottom for final comments.

Role Model and Permissions:

Administrator: Has an interface to upload final ZIP archives and distribute them into the cards of the respective ad networks.

Client: Sees only the download interface. Selects the needed networks via checkboxes and presses the "Download selected builds" batch download button.
