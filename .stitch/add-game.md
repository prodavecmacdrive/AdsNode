Screen #2 Specification: Game Project Creation Questionnaire (Wizard)

Display format: Popup (modal window) overlaying the main page

Step 1. Game Identification

Main field: Large input for pasting an App Store or Google Play link.

Auto-parsing: After pasting the link, the system automatically pulls the official icon and game title, displaying them in the preview window.

Manual entry: Toggle button "Game is not in the store yet". When clicked, an area opens to upload an icon from the computer and a text field for manual title entry.

Role Model and Permissions:

Administrator: Full access to the form (pasting store links, uploading icons, manual entry).

Client: Page is completely inaccessible. Attempting to navigate via a direct link results in a redirect to index.md.
